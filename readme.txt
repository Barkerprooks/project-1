How to compile:
  $ cd project-1
  $ make

How to run:
  ./flights.$(uname -m)
  or
  ./flights<tab> + <enter>

How to use:
  the main menu will show once 
  the program is run. it will
  show after each option selected
  unless the option is q

  1 - runs task 1
  2 - runs task 2
  3 - runs task 3
  l - lists all nodes
  i - lists nodes with connections
  q - quits program gracefully
